# let-s_booking
🏨 naverMap API를 사용한 숙소예약앱 🏨

## 개발환경

* OS : windows 10 pro
* IDE : Android Studio
* Language : Kotlin

## 사용한 것

* Naver Map API
* ViewPager2
* FrameLayout
* CoordinatorLayout
* BottomSheetBehavior
* Retrofit2
* Glide
